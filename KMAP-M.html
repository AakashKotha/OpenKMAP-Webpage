KMAP-M Toolbox Overview

The KMAP-M Toolbox is an advanced MATLAB toolkit, building upon the core C/C++ functionalities of KMAP-C. It provides a user-friendly MATLAB interface for implementing and analyzing tracer kinetic models, simplifying dynamic PET data workflows. Designed to be flexible and efficient, it operates across Windows, Linux, and Macintosh systems.

Features and Directory Structure:

  • Directory Structure:
    • demo: Demonstrations and test cases to guide users in employing kinetic modeling approaches effectively.
    • fit: Contains MATLAB wrapper functions for kinetic modeling, focusing on fitting compartmental models like 1TCM, 2TCM, and the liver model.
    • tac: Code for generating Time-Activity Curves (TACs) from input functions, offering flexibility across various kinetic models.
    • data: Example datasets to facilitate testing and demonstration of the provided tools.
    • utils: A collection of utility functions, such as those for decay correction and plasma-to-blood ratio calculations, to support modeling workflows.
    • Precompiled_Binaries: Contains precompiled MEX binaries for TAC fitting. For custom setups, these can be recompiled using KMAP-C.
  • Supported Models and Functions:
    • One-Tissue Compartment Model (1TCM): Analyze simple tracer kinetics with parameters for blood volume and transfer rates.
    • Two-Tissue Compartment Model (2TCM): A more comprehensive model for reversible binding kinetics.
    • Liver Model: Optimized for hepatic tracer dynamics, offering a seven-parameter approach.
    • Patlak Plot Analysis: A graphical method for quantifying tracer influx rates.
    • Relative Patlak Plot: A variant of the Patlak plot for scenarios lacking early-time input functions.

How to Start

To get started, run the setup.m file in the main directory to add all the required paths to MATLAB. Detailed usage examples and test cases are provided in the demo folder. For advanced instructions, please visit the KMAP-M GitHub Repository.

Why KMAP-M?

  • Simplifies high-performance kinetic modeling within MATLAB.
  • Provides a direct bridge to C/C++ implementations for computational efficiency.
  • Encourages community-driven development and research by being open-source.

About KMAP-M

The KMAP-M Toolbox is a MATLAB-based extension of the KMAP ecosystem, designed for tracer kinetic modeling, time-activity curve generation, and advanced parametric imaging.
