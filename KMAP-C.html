KMAP-C Toolkit Overview

The KMAP-C Toolkit is the computational core of the OpenKMAP ecosystem, specifically designed for developers working with dynamic PET data. It provides a performance-optimized C/C++ source code library tailored for tracer kinetic modeling, focusing on efficiency, extensibility, and integration with high-level programming languages.

Features and Directory Structure

Directory Structure:

  • src Folder: Contains the core implementations for tracer kinetic models, input function definitions, optimization algorithms, and utility functions. This is where the backbone of the toolkit resides, enabling researchers to implement custom kinetic models.
  • mex Folder: Includes precompiled binaries and wrapper functions to integrate KMAP-C with MATLAB seamlessly. These wrappers allow MATLAB users to utilize the efficiency of the C/C++ implementations without needing to interact with the core C/C++ code.

Supported Models:

  • One-Tissue Compartment Model (1TCM): Used for simple tracer kinetics, fitting parameters like blood volume and transfer rates.
  • Two-Tissue Compartment Model (2TCM): Provides a more complex tracer analysis by introducing additional parameters for reversible binding sites.
  • Liver Model: A specialized seven-parameter model tailored for analyzing tracer dynamics in hepatic tissues.

Interoperability and Efficiency

OpenMP Acceleration: The toolkit uses OpenMP for parallel processing, significantly enhancing the performance of voxel-wise kinetic modeling for total-body PET data.

Cross-Language Support: MATLAB users can directly use the KMAP-C capabilities via provided MEX wrappers. Future releases aim to extend this support to Python, broadening accessibility for developers and researchers.

Documentation and Support

To get started, visit the KMAP-C GitHub Repository for comprehensive documentation, code samples, and setup instructions. Whether you're implementing advanced kinetic models or conducting large-scale parametric imaging, KMAP-C is designed to save time and streamline the development process.
