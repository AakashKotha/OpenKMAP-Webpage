OpenIFun: Innovating Input Functions in Dynamic PET

OpenIFun is an ambitious initiative within the OpenKMAP framework, designed to tackle challenges in acquiring reliable blood input functions for tracer kinetic modeling in dynamic PET imaging. Currently in its implementation phase, this project aims to develop a robust database for researchers and practitioners in molecular imaging.

Key Highlights:

  • Significance:

    OpenIFun is focused on enabling innovative techniques in kinetic modeling and parametric imaging. It addresses critical challenges, such as deriving accurate population-based input functions and training AI algorithms for molecular imaging.

  • Proposed Database:
    • OpenIFun4FDG: This initial project within OpenIFun aims to collect image-derived input functions (IDIFs) from over 200 human subject scans using total-body PET scanners like the UIH uEXPLORER and Siemens Quadra.
    • These datasets are designed to support the development of models and algorithms, including population-based input function models and optimization techniques for brain PET imaging.
  • Potential Applications:
    1. Building population-based models for whole-body Patlak parametric imaging.
    2. Supporting shortened dynamic scans that do not capture early dynamic phases.
    3. Facilitating dynamic brain PET imaging with enhanced optimization methods.
    4. Training deep-learning models using paired datasets of high- and low-quality IDIFs.

About OpenIFun

OpenIFun is an open-data initiative under development, aiming to build a comprehensive database of blood input functions to support tracer kinetic modeling in molecular imaging using dynamic PET.
